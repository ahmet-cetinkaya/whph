• Нова опція фільтра для легкого відображення або приховування підзавдань для кращого управління завданнями
• Покращений експорт журналів налагодження з повною підтримкою мов для більш плавного усунення несправностей
• Покращені журнали налагодження тепер відображаються як накладені сповіщення для миттєвого зворотного зв'язку
• Повна функція експорту журналів з локалізацією для всіх підтримуваних мов
• Додано переклади помилок для журналів налагодження кожною мовою для покращення користувацького досвіду
• Введено розширені налаштування та повну функціональність журналів налагодження для досвідчених користувачів
